Donnybrook

Donnybrook is a SpeedRun Race organization bot. (I'm still working on him, got some bugs to fix first)
Want to use Donnybrook on your Discord, then click below to invite Donnybrook to your discord, or if you need help, click the other button to join the Donnybrook Discord.
Already running Donnybrook? and need a reference for the commands click commands up in the corner.
Donnybrook is created by MediaMagnet and is written in GoLang, and uses mongoDB.
