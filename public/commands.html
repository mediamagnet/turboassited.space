Commands:

Donnybrook has the following commands:
 
Command
 
Description
 
.setup Game, Category
This will create a new race inside donnybrook and create a text and voice channel with the race id as its name.
Example .setup A link to the Past, Any%
 
.join
Join a race with the specified id
example: .join ABCD
 
.leave
Leave a race with the specified id before it is started
example .leave ABCD
 
.ready
Ready up for the race after you're done setting up
 
.unready
Leave the ready state for the race.
 
.start
Once everyone is ready start the race.
 
.done
Once you finish the race use this. Donnybrook will give you your total time in the race.
 
.forfeit
Not able to keep going use this to exit the race early.
 
.help
You're reading it.
 
.invite
Got a question? join the Donnybrook discord here: https://discord.gg/cyZzPZY
 
.voicejoin
Join a voice channel for voiced countdown. Must be in voice channel for this to work.
 
.voiceleave
Leave the voice channel
 
.cleanup
Cleans messages in channel command is run in. User must have Manage Message permissions.
 
.scatter "Channels to scatter to"
Scatter users to the provided channels. User must have Manage Server Permissions.
Example: .scatter ChannelA,ChannelB,ChannelC
